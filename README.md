How to use this app:

1. Create your telegram-bot by using [@BotFather](https://t.me/botfather) and sending /newbot command.
2. Find out your telegram user ID by using any services for it. Just Google it :)
3. Copy and paste token and your user ID obtained above to appropriate text fields and click "save" button.
4. Now your app is ready to forward SMS to your telegram bot!
